Fun things to do with MacOS

Just a list of fun tools I ran across. Will keep adding to it as I figure out more.

say

It's a simple command that you can use to let your Mac speak.

  • say -v ?
    • Shows you a list of possible voices (with accents!!)
  • say -v Xander Guten Tag!
  • say -v Karen Hey mate, what's up!

    • Make some noise

  • afplay "/Applications/iMovie.app/Contents/Resources/iMovie Sound Effects/Chicken.mp3"


    • Apple Script

    I haven't really had time to dive into Apple Script, just a seeing how it can be used for. :) These few things I found already quite useful:

    Running scripts via osascript

  • osascript -e 'set volume output volume 100'"

    • Change the volume, useful to make sure say can be heard.

    Spoofing a password prompt

  • VALUE=$(osascript.exe -e 'display dialog "Please enter your password"')

    • Showing Notifications

  • osascript -e 'display notification "Logon detected with account macosuser from 192.168.0.3. Is this expected?" with title "[Notification]" '
